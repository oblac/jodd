    Jodd

    The Unbearable Lightness of Java

    Github Stack Overflow Contact us Twitter Users Search

    Jodd is set of Java micro frameworks, tools and utilities, under 1.7 MB.

    We believe in common sense to make things simple, but not simpler.

    Get things done! Make your ideas! Kickstart your startup!

    And enjoy the coding.

    Jodd table of Micro-components

    Here are some components for you! See documentation for more.
    B
    BeanUtil - fast bean library, type introspection and type conversion tools with some unique features.
    T
    JDateTime - astronomical precision in single time-manipulation class.
    R
    Props - properties replacement, featuring: UTF8, sections, profiles, macros and more.
    D
    DbOom - thin database layers, object mapper and Template-SQL that simplifies database code.
    M
    Madvoc - MVC framework based on CoC and annotations for pragmatic web development.
    P
    Petite - DI container that supports sufficient most of features offered by other containers.
    X
    Proxetta - the fastest proxy creator with new, unique approach for making pointcuts and different type of advices.
    L
    Lagarto - all-purpose HTML parser and DOM builder.


    J
    Jerry - jQuery-friendly HTML parser with CSS3 selectors support.


    H
    HTTP - Tiny and raw HTTP client, helps talking to servers.

    E
    Email - sending and receiving emails for busy developers.

    C
    Decora - decoration framework for web pages based on templates.

    S
    Json - parsing JSON to objects and serializing objects back to JSON.

    A
    HtmlStapler - auto packaging of web resources included by HTML page.
